# DevSecOps-Pro
Notes for future / Exam
>Create basic security report
```bash
cat > security-report.json <<EOL

{
    "Vulnerabilities": "10",
    "Valid": "2",
    "Severity": "High",
    "generated_at": "2022-05-16T17:37:50Z"
}

EOL
```

>View file 
```bash
vi security-report.json
```
> create a foo.txt file
```bash
vi foo.txt
```

```bash 
/High
```
